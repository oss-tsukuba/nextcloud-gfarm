# INSTALL

1. Copy ./html/custom_apps/files_external_gfarm into html/custum/apps/
2. Copy ./bin/* into $HOME/www/bin/
3. From WebUI, enable External Storage in Apps
4. From WebUI, enable External Storage Gfarm in Apps
5. From WebUI, check "Allow users to mount external storage Gfarm" in Settings>Administration>External Storage
